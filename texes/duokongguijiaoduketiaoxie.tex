\begin{frame}[c]
    \frametitle{基于多孔硅可调谐光学滤光片的微光谱仪：原理和制造过程}
    \begin{description}
        \item[原理] 通过调节多孔硅的孔隙率可以改变其折射率，以此来制作反射镜形成法布里-珀罗滤光片。通过调节机械臂的角度改变光的入射角度，从而调节峰值波长。
    \end{description}

    工艺流程：
    \begin{itemize}
        \item $\mathrm{Si}_3\mathrm{N}_4$ 掩膜确定硅基底上多孔化和电抛光的区域
        \item 电化学刻蚀（氢氟酸和乙醇）形成多孔硅
        \item 电解抛光
        \item 在空气中晾干
        \item 充分氧化
    \end{itemize}
